The basics

Nextcloud is a cloud service just like Google Drive, OneDrive and iCloud etc. The big difference is that this cloud is not hosted in someone else's datacenter. This cloud is hosted by you, controlled by you and free to use.

Crucial features

Nextcloud has all the features I deem necessary in a cloud backup enviroment. There are desktop and mobile applications so you can synchronize all your devices. I use this on my desktops so all my crucial files and folder are synced between all my devices. The files get uploaded, downloaded and updated to all the devices and the server whenever the devices have an internet connection. On my mobile device it synchronizes the camera gallery to Nextcloud. This way I don't have to worry about my precious memories getting lost if something happens to my phone.

Administrator features

- Two-factor verification
I have enabled this like I believe everybody should do with as much as possible.
-Email notifications
To make sure NextCloud stays up to date for the best security practice it sends an email whenever there is an update.

Additional features

NextCloud has an apps feature which allows you to add a lot of things. Some things I added and use regularly:
  • Maps, to see the location of contacts and where pictures were taken.
  • Talk, similar to MS Teams to communicate with people.
  • Notes, a simple and convenient note taking app for anything from groceries to development tasks.
